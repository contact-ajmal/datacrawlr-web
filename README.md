# Datacrawlr — Marketing Website (`datacrawlr-web`)

This repository contains the standalone marketing website for **Datacrawlr**, the dataset and model intelligence layer. It is built using **Next.js 15**, **TypeScript**, **Tailwind CSS v4**, **shadcn/ui**, and **Framer Motion**.

---

## 🚀 Deployment

This site is optimized and pre-configured for zero-friction edge deployments on **Cloudflare Pages**.

### Cloudflare Pages Requirements:
- **Compatibility Date:** `2024-09-23` or later.
- **Compatibility Flags:** `nodejs_compat` enabled (pre-configured via `wrangler.toml`).

---

## 🛠️ Local Development

First, run the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
